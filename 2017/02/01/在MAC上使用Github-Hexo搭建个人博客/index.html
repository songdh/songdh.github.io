在MAC上使用Github+Hexo搭建个人博客

Feb 1, 2017 | 前端 | 阅读

曾经在Github上用Jekyll搭建过博客，但是一直没有更新过。后来无意看到一篇帖子，是用Hexo来搭建个人博客。大概看了下，操作命令非常简单，于是就动手使用一下Hexo。博客使用markDown语法，生成和上传非常简单，还能支持本地预览，简直爱不释手。

参考

  • Hexo
  • Hexo搭建个人博客

Hexo是什么

Hexo是一个快速、简单、强大的博客框架。可以使用Markdown来编写博客，然后Hexo可以将它生成漂亮的静态页面。

安装要求

  • Xcode
  • Node.js
  • Git

安装Xcode时会自动安装Command Line Tools，很多开发工具都会依赖于它。在mac上做开发，Xcode是必须要安装的。

For Mac users
You may encounter some problems when compiling. Please install Xcode from App Store first. After Xcode is installed, open Xcode and go to Preferences -> Download -> Command Line Tools -> Install to install command line tools.

Hexo使用Nodejs来生成静态页面，使用Github来管理博客。
网上很多帖子介绍了很多种方法来安装nodejs和Git，看得我都凌乱了。所以其他的方法都不要，按照官方文档，只要点击上面的链接，下载安装就行了。

帐号配置完成后，要创建一个GitHub Pages的仓库，下面记录下创建过程

  1. 在页面上点击New按钮，创建一个新的仓库

    仓库名称上，最好按照图中的写法xxx.github.io,这样最后生成的页面就可以使用https://xxx.github.io来访问了，如果名称写的是songdh，那么最终的页面地址会是https://songdh.github.io/songdh/

  2. 点击setting进入设置页面

  3. 在设置界面向下滚动到GitHub Pages一项，由于我们的仓库中任何文件都没有，所以显示页面不可用。
    我们可以选择一个主题，让界面生效。按照图中所示，选择一个主题。

  4. 在浏览其中访问https://songdh.github.io,查看当前效果
